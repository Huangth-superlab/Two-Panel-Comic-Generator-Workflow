# 双句漫画自动生成工作流 - 开发手册

## 📖 项目概述

**项目名称**：双句漫画自动生成工作流（Comic Generator Workflow）

**项目简介**：基于LangGraph框架开发的智能漫画生成系统，能够根据用户提供的上下句文案，自动生成具有治愈系风格的彩绘铅笔风格双格漫画。

**核心功能**：
- 智能情绪分析（支持15种情绪类型识别）
- 场景理解与描述生成
- 彩绘铅笔风格漫画生成
- 自动添加文案文字
- 对象存储与URL返回

**项目版本**：v18.0

---

## 🛠 技术栈

### 核心框架
- **LangGraph 1.0.2**：工作流编排框架，构建DAG图结构
- **LangChain 1.0.3**：大语言模型调用与工具链
- **Pydantic 2.12+**：数据验证与类型管理

### AI能力
- **coze-coding-dev-sdk**：大模型、图片生成、对象存储等集成能力
- **豆包大模型（doubao-seed-2-0-lite-260215）**：场景分析与情绪识别

### 图像处理
- **Pillow 10.3+**：图片处理与文字添加
- **OpenCV 4.12+**：图像处理辅助工具

### 数据存储
- **boto3 1.40+**：AWS S3兼容的对象存储SDK
- **S3SyncStorage**：对象存储上传与管理

### 工具库
- **Jinja2 3.1+**：模板引擎（提示词渲染）
- **requests 2.32+**：HTTP请求
- **python-dotenv 1.2+**：环境变量管理

│   ├── agents/                          # Agent代码（当前为空）

│   │   ├── graph.py                     # 主图编排
│   │   └── nodes/                       # 节点实现

│   │       ├── scene_analysis_node.py   # 场景分析节点
│   │       ├── prompt_assemble_node.py  # 提示词组装节点
