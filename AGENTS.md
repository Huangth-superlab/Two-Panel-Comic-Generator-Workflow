## 项目概述
- **名称**: 双句漫画自动生成工作流
- **功能**: 基于上下句文案自动生成双格治愈系漫画，使用脚本添加文字

### 节点清单
| 节点名 | 文件位置 | 类型 | 功能描述 | 分支逻辑 | 配置文件 |
|-------|---------|------|---------|---------|---------|
| scene_analysis | `graphs/nodes/scene_analysis_node.py` | agent | 场景分析与描述生成（合并关键因素提取和场景描述） | - | `config/scene_analysis_llm_cfg.json` |
| prompt_assemble | `graphs/nodes/prompt_assemble_node.py` | task | 提示词组装（Python逻辑） | - | - |
| image_gen | `graphs/nodes/image_gen_node.py` | task | 图片生成（带参考图） | - | - |
| add_text | `graphs/nodes/add_text_node.py` | task | 添加文字（脚本） | - | - |

**类型说明**: task(task节点) / agent(大模型) / condition(条件分支) / looparray(列表循环) / loopcond(条件循环)

## 子图清单
无子图。

## 技能使用
- 节点`scene_analysis`使用大语言模型技能和web-search技能
- 节点`prompt_assemble`不使用技能（Python逻辑）
- 节点`image_gen`使用图片生成技能和对象存储技能
- 节点`add_text`使用对象存储技能

## 工作流说明

### 输入参数
- `upper_sentence`: 上句文案内容（必填）
- `lower_sentence`: 下句文案内容（必填）
- `explanation`: 解释说明（可选，用于补充上下文信息，影响关键因素生成）

### 处理流程
1. **场景分析与描述生成**（合并关键因素提取和场景描述）
   - 同时理解上下句文案
   - 分析句子的显著特征和关键词
   - 识别句子想表达的情绪类型：
     * 反讽、反差、惊讶、摆烂、治愈、励志
     * 搞笑、温馨、荒诞、治愈系无厘头、怀旧、离别、困惑、顿悟
   - 生成详细的场景描述（中文）
   - 主角固定为参考图中的粉色小猪
   - 可以生成类似的配角小猪
   - 确定两格关联关系（对比/递进/因果/并列）
   - **保留句子中的具体事物名词、物品、人物、地点等**
   - **如果有解释说明，会参考解释中的上下文信息**
   - 保持无厘头、荒诞和戏剧性的表现力
   - 角色动作和表情要夸张、生动、充分体现情绪
   - 背景风格必须与角色一致（2D扁平卡通风格）

3. **提示词组装**（Python逻辑）
   - 使用固定模板
   - 将场景、句子填充到模板
   - 不使用大模型，直接字符串组装
   - 明确要求模型不生成文字，预留空白区域

4. **图片生成**
   - 使用组装好的提示词
   - 从本地读取角色设计图和框体模板
   - 上传到对象存储后作为参考图传入
   - 生成双格漫画（3:4垂直比例）
   - 彩绘铅笔风格，具有真实感和高级艺术感
   - 禁用水印
   - 不生成文字，预留空白区域

5. **添加文字**（Python脚本）
   - 下载生成的图片
   - 使用Pillow库添加文字
   - 上格添加上句文案
   - 下格添加下句文案
   - 文字居中显示，带白色边框

### 输出结果
- `comic_with_text_url`: 添加文字后的漫画图片URL

### 风格特点
- **整体风格**: 治愈系、温暖、可爱、彩绘铅笔风格
- **色调**: 粉白、淡黄色调，色彩细腻柔和
- **角色**: 卡通小猪（粉色，圆润身体），彩色铅笔绘制
- **格式**: 双格垂直漫画（3:4比例）
- **细节**: 彩色铅笔质感、笔触细腻、具有真实感和高级艺术感
- **水印**: 无水印
- **文字**: 使用脚本添加，黑色粗体字，带白色边框，居中显示
